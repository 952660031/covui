### Message

* `message` \- message text
* `duration` \- display duration, seconds
* `onClose` \- callback function when closed with the message instance

|  propName  | propType | defaultValue | acceptedValue |
| ---------- | -------- | ------------ | ------------- |
| message    | string   | -            | -             |
| duration   | number   | 3            | -             |
| onClose    | function | null         | -             |
